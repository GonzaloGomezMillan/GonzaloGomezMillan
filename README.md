## About Me

Hi there! I'm Gonzalo Gomez-Millan del Villar, a passionate data engineer with experience in AWS and GCP. I have a background in both batch and streaming processes and currently working on a project with GCP to improve energy management. I love learning new technologies and staying updated with the latest trends in the industry. This repository serves as a showcase of my personal projects and contributions to the open-source community.
